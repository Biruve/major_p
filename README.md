# AI-Driven Natural Language to SQL Query Generation

## Project Overview
This project converts natural language questions into SQL queries using a transformer-based deep learning model (FLAN-T5). The generated SQL is executed on a MySQL database and the results are displayed through a simple web interface.

## Technologies
- Python
- FLAN-T5
- Transformers
- PyTorch
- MySQL
- Streamlit

## Features
- Natural language to SQL conversion
- Real-time database querying
- User-friendly interface
- SQL validation before execution

## Note
Due to repository size limitations, the trained model weights and dataset are not included. They can be recreated using the provided scripts.
